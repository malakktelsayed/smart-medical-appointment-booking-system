# 🏥 Smart Medical Appointment Booking System

A healthcare management system built with **Python** and **MySQL**. The application enables patients to book appointments, manage medical records, and allows doctors and administrators to efficiently manage healthcare services through a relational database.

---

## 📌 Features

- Patient registration and login
- Doctor management
- Appointment booking and cancellation
- Clinic scheduling
- Medical reports management
- Insurance management
- Patient reviews and ratings
- Database normalization (3NF)
- SQL constraints and relationships

## 🗄️ Database Design

The database was designed using Entity-Relationship (ER) modeling and normalized to Third Normal Form (3NF) to reduce redundancy and maintain data integrity.

## 🚀 How to Run

1. Install Python 3.x.
2. Install MySQL Server and MySQL Workbench.
3. Import the SQL file from the `sql` folder.
4. Update the database connection settings in `app.py` if needed.
5. Run the application:

```bash
python app.py
```
